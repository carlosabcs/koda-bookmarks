# Privacy Policy for Koda Bookmarks

**Effective Date:** May 1, 2026

This Privacy Policy applies to the "Koda Bookmarks" browser Extension ("the Extension").

## 1. Data Collection and Usage

Koda Bookmarks is designed with strict privacy in mind. **The Extension does not collect, store, transmit, or share any personal data, browsing history, or bookmark data with any external servers or third parties.** All data processing happens entirely locally on your device.

## 2. Permissions Justification

To function properly, the Extension requires the following permissions, which are strictly used locally:

- **Bookmarks:** Used exclusively to read your existing bookmark tree to provide search functionality, and to create, modify, or delete bookmarks at your direct request. Your bookmarks never leave your browser.
- **ActiveTab / Host Permissions:** Used exclusively to read the title and URL of the currently active tab when you open the extension, allowing you to quickly save the current page.
- **Scripting:** Used strictly to inject the extension's user interface into the current webpage.

## 3. Third-Party Services

The Extension does not use any third-party analytics, tracking software, or external APIs.

## 4. Open Source

Koda Bookmarks is fully open-source. Anyone can review the source code to verify that no malicious activity or hidden data collection is occurring.

## 5. Contact

If you have any questions about this Privacy Policy, please open an issue on our GitHub repository.
